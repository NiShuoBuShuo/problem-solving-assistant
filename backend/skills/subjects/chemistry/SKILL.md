## 化学解题规范

**标准步骤流**：
题型识别 → 反应分析 → 已知信息 → 方程式写出与配平 → 计算推导 → 最终答案 → 验证 → 知识点总结

**方程式规范**：
- 化学方程式必须配平（左右原子数相等）
- 用 LaTeX 表示化学式（例：$\text{H}_2\text{O}$、$\text{CO}_2$、$\text{H}_2\text{SO}_4$）
- 反应条件标在等号上方：$\overset{\Delta}{=}$、$\overset{\text{点燃}}{=}$
- 可逆反应用 $\rightleftharpoons$，不可逆反应用 $\rightarrow$
- 沉淀用 $\downarrow$，气体用 $\uparrow$

**摩尔计算**：
- 步骤：写化学方程式 → 找已知量与目标量的摩尔比 → 计算摩尔数 → 换算质量/体积/粒子数
- 公式：$n = \dfrac{m}{M}$，$n = \dfrac{V}{V_m}$（标准状况下 $V_m = 22.4\ \text{L/mol}$），$n = \dfrac{N}{N_A}$

**实验题**：
- 按"现象 → 分析 → 结论"三段结构
- 实验操作步骤：序号清晰，每步一个操作
- 安全注意事项：酸碱稀释顺序（酸入水）、加热操作等

**离子方程式**：
- 强电解质拆开，弱电解质/难溶物/气体保留分子式
- 检验电荷守恒

**有机化学**：
- 写出官能团，说明其决定的化学性质
- 同分异构体：系统列举（碳链/官能团位置异构）
