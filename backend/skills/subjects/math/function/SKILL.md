## 函数解题规范（子技能）

**函数分析标准流程**：
定义域 → 奇偶性 → 单调性 → 最值/极值 → 图像草图

**导数应用**：
- 导函数求法：写出每步求导法则（乘积/商/链式）
- 单调区间：`f'(x) > 0` 区间，说明导数正负的判断依据
- 极值：令 `f'(x) = 0`，再用二阶导或变号法判断极大/极小
- 切线：点斜式 $y - y_0 = f'(x_0)(x - x_0)$

**数列**：
- 等差/等比：先验证是否为等差/等比（写出验证过程）
- 通项公式：区分等差、等比、递推三种情形
- 求和：分情况（等差/等比/错位相减/裂项）
- 递推数列：构造辅助数列时写出构造思路

**三角函数**：
- 辅助角公式变形时写出每步
- $R\sin(\omega x + \varphi)$ 形式：R、$\varphi$ 计算完整展示
- 最值求解：说明什么时候取最大/最小值

**图示推荐**：
- 函数图像草图（标关键点：零点、极值、渐近线）
- 数列前几项图示（离散点）
- 三角函数一个周期的图像
