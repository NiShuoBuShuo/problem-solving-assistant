## 语文解题规范

**标准步骤流**：
题型识别 → 审题 → 原文定位与引用 → 分析（结构/修辞/情感/主旨）→ 组织答案 → 答题模板说明

**原文引用（必须执行）**：
- 所有分析必须引用原文句子，标注为题内依据（`in_problem`）
- 严格区分"原文说了什么"与"你的分析"
- 引用原文后紧跟分析，不要只复制原文

**现代文阅读**：
- 概括类：先找关键句（段首/段尾/转折后），再归纳
- 分析表达手法：指出手法名称 + 结合原文分析效果 + 表达情感/主旨
- 修辞题：比喻/拟人/排比/对比—写出表达效果
- 主旨题：结合全文（不能以偏概全），关注标题、开头、结尾

**古诗文**：
- 翻译：直译为主，意译为辅；逐词落实（实词/虚词）
- 手法：意象分析 → 意境描述 → 情感归纳
- 炼字题：释义 → 手法（拟人/通感/化静为动）→ 效果
- 古文断句：找主谓宾，找语气词（也/矣/哉/乎）

**答题结构**：
- 分条列点（不要写成一大段）
- 每条先写结论，再写依据（总-分结构）
- 非唯一答案：给出依据链条，说明组织逻辑

**作文**：
- 审题：找出限制词、核心词、关系词
- 立意：明确主旨（一句话表达）
- 结构：开篇点题 → 论证 → 结尾升华
