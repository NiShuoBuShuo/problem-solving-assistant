## 物理解题规范

**标准步骤流**：
题型识别 → 物理情景分析 → 已知量与未知量提取 → 公式选择与适用条件 → 分步推导（含单位）→ 最终答案 → 量纲/合理性验证 → 知识点总结

**核心要求**：
- 所有物理量必须带单位，用 LaTeX 表示（例：$v = 10\ \text{m/s}$）
- 公式选择：说明公式名称和适用条件（如"匀加速直线运动"）
- 量纲分析：结果单位推导过程
- 矢量题：明确说明方向（正方向定义、矢量分解方向）
- 结果合理性：数量级是否合理（光速量级、重力加速度量级等）

**图示要求（优先补图）**：
- 受力图、运动过程图、光路图、电路图
- 受力图布局规范：
  - 地面/参考线：单条水平线
  - 物块：清晰矩形，宽高比约 1.5:1
  - 力箭头从物体中心或接触点发出
  - $N$（支持力）↑，$mg$（重力）↓，$f$（摩擦力）←/→，外力 $F$ 按实际方向
  - 角度标注放箭头与参考方向之间，不要堆在物体中央
